#HTML 块

HTML块元素

块元素在显示的时候，通常会以新行开始
如：<h1>、<p>、<ul>

HTML内联元素

内联元素显示通常不会以新行开始
如<b>、<a>、<img>

HTML <div>元素

<div>元素也被称为块元素，其主要是组合HTML元素的容器,配合css样式使用

HTML <span> 元素

<span>元素是内联元素，可作为文本的容器
