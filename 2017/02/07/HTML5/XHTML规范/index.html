闲暇时间学习学习HTML相关知识————XHTML规范！

XHTML 介绍

  1. 什么是XHTML?
    XHTML指的是可扩展的超文本标记语言
    XHTML与HTML 4.0.1几乎是相同的
    XHTML是更严格更纯净的HTML版本
    XHTML是以XML应用的方式定义的HTML
    XHTML得到了主流浏览器支持

  2. 为啥使用XHTML？
    为了代码的完整性和良好性

  3. 文档声明
    DTD：规定了使用通用标记语言的网页语法
    类型：

    STRICT（严格类型）
TRAMSOTOPMAL（过渡类型）
FRAMESET（框架类型）

XHTML 元素

  1. XHTML 元素必须正确嵌套
  2. XHTML 元素必须始终关闭
  3. XHTML 元素必须小写
  4. XHTML 元素必须有一个根元素

XHTML 属性

  1. XHTML 属性必须使用小写
  2. XHTML 属性值必须用引号包围
  3. XHTML 属性的最小化也是静止的
