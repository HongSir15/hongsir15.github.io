闲暇时间学习学习HTML相关知识————HTML列表！

HTML 列表

有序列表　　　<ol>
无序列表　　　<ul>
列表项　　　　<li>
列表　　　　　<dl>
列表项　　　　<dt>
描述　　　　　<dd>

示例

分类

  1. 无序列表
    使用标签：<ul>、<li>
    属性：disc、circle、square

  2. 有序列表
    使用标签：<ol>/<li>
    属性：A、a、l、i、start (type取值)

  3. 嵌套列表
    使用标签：<ul>/<ol>/<li>

  4. 自定义列表
    使用标签：<dl>/<dt>/<dd>
